## 👨‍💻 About
This repository documents my complete C++ learning journey as a Computer Engineering student at LJ University. Each folder represents a topic with clean, well-commented code and notes.

## 📂 Structure
```
cpp-journey/
├── 01-basics/          # Variables, data types, operators
├── 02-control-flow/    # Loops, conditions, switch
├── 03-functions/       # Functions, recursion, scope
├── 04-oops/            # Classes, objects, inheritance
├── 05-stl/             # Vectors, maps, algorithms
├── 06-data-structures/ # Linked lists, stacks, queues
├── 07-algorithms/      # Sorting, searching
└── projects/           # Mini projects
```
